# Our commitment to security
# We take security seriously and will respond to valid reports within 72 hours.
# We appreciate responsible disclosure and ask that you:
# - Allow us reasonable time to respond before public disclosure
# - Avoid accessing or modifying user data
# - Report vulnerabilities directly through the contact above

# Scope
# The following are in scope for security reports:
# - Authentication/authorization bypass
# - Remote code execution
# - SQL injection (though we use KV store)
# - Cross-site scripting (XSS)
# - Cross-site request forgery (CSRF)
# - Server-side request forgery (SSRF)
# - Information disclosure
# - Privilege escalation

# Out of scope:
# - Denial of Service attacks
# - Social engineering
# - Physical attacks
# - Attacks requiring physical access to a user's device
# - Issues in third-party services not under our control

# Recognition
# We appreciate security researchers who help keep our users safe.
# While we don't currently offer a bug bounty program, we will acknowledge
# responsible reporters in our security hall of fame (with permission).
